# DealClaw 外贸数字员工平台 - Demo

> 内测版 v2.0 - AI驱动的外贸获客解决方案

## 🎯 项目简介

DealClaw 是一款面向外贸 B2B 企业的 AI 数字员工协作平台，帮助企业自动化获客、客户培育和增长分析。

## ✨ 核心功能

### 1. 任务看板 (Dashboard)
- 实时统计指标（新增线索、客户互动、任务进度）
- 快速创建获客任务（自然语言输入）
- 任务列表与进度追踪
- 最近动态通知

### 2. NEO 获客工作台
- AI 获客专员自动执行客户筛选
- 多需求并行管理（最多3个）
- 客户 A/B/C 三级分级
- 实时进度追踪

### 3. 客户管理
- 客户列表与详情查看
- 采购意向分析
- AI 评级与建议话术
- 开发信发送状态追踪

### 4. WhatsApp 集成
- 直接与客户 WhatsApp 沟通
- 消息历史记录
- 快捷回复模板
- 在线状态显示

### 5. 自动开发信
- AI 根据客户信息生成个性化邮件
- 邮件主题和内容可编辑
- 一键发送/重新生成

## 🚀 快速开始

### 本地运行

```bash
# 进入项目目录
cd dealclaw-demo

# 启动本地服务器
python3 -m http.server 9999

# 浏览器访问
open http://localhost:9999
```

或直接双击 `index.html` 在浏览器中打开。

## 🎯 使用指南

### 创建获客任务
1. 在 Dashboard 输入自然语言描述（如："帮我找50个美国LED批发商"）
2. AI 自动解析配置
3. 确认后 NEO 开始执行

### 查看客户详情
1. 进入 NEO 工作台
2. 点击"查看客户"
3. 查看采购意向、AI建议话术、评级原因

### 发送开发信
1. 在客户列表点击"写开发信"
2. 查看 AI 生成的个性化邮件
3. 编辑或直接发送

### WhatsApp 沟通
1. 点击客户旁的 WhatsApp 图标
2. 查看聊天历史
3. 使用快捷回复或自定义消息
